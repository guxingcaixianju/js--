1.认识原型链

原型链是指每个对象的原型都指向一个上层的原型对象，直到遇到null，js中最顶层的一个对象是Object，
它的原型是object.prototype，但是Object.prototype.__proto__ 是null。

2.原型链访问原理

访问一个对象的属性时，会先在自身属性中查找，找到返回；如果没有找到，则在自己的__proto__属性中找，
如果找到，则返回，否则在构造函数的原型对象中找，如果没有找到，则再沿着__proto__这条链向上查找，
找到返回，找不到，返回undefined。

4.理解prototype和__proto__

function User(){}
var obj = new User();
User有两个属性：prototype和__proto__,其中prototype是服务于User创建的实例对象的,
由它创建的实例化对象是指向prototype的，如下验证所示：
console.log(User.prototype == obj.__proto__);
当将User当做对象使用时，用它的__proto__属性。
User.__proto__.view = function(){
console.log("nice");
}

创建一个数组对象arr,查看arr的__proto__属性，它指向Array的prototype对象
let arr = ["a"];
console.log(arr.concat("b"));
console.dir(arr);
console.log(arr.__proto__ == Array.prototype);
创建一个一般对象，查看它的__proto__属性，指向Object的prototype对象
